# Árvores AVL e Rubro-Negra

Projeto desenvolvido para comparação entre Árvores AVL e Árvores Rubro-Negras em operações de:

- Inserção
- Remoção
- Busca
- Balanceamento automático

O objetivo é analisar o desempenho das estruturas em cenários de grande volume de dados.
